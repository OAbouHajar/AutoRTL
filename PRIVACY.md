# Privacy Policy — AutoRTL

**Last updated:** April 15, 2026

## Data Collection

AutoRTL does **not** collect, store, or transmit any personal data or browsing information.

## What the extension does

- Reads text content on web pages **locally** to detect Arabic script and apply RTL text direction.
- Stores your preferences (enabled state, direction mode, selected font) **locally** on your device using `chrome.storage.local`.

## What the extension does NOT do

- Does not collect personally identifiable information.
- Does not track browsing history or user activity.
- Does not send any data to external servers.
- Does not use analytics, cookies, or tracking scripts.

## External Resources

When you select a custom Arabic font, the extension loads a CSS stylesheet from Google Fonts (`fonts.googleapis.com`). No user data is sent to Google — only a standard font request is made.
